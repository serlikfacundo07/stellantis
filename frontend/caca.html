Plan de Apertura
Lista ordenada por urgencia de los contenedores a abrir. La columna Prior. indica el orden recomendado:
Prioridad 1–3 (rojo): apertura inmediata, stock crítico.
Prioridad 4–7 (amarillo): apertura en las próximas horas.

Importación de Stock
El sistema acepta datos desde dos fuentes independientes:
Fuente A — Stock de Planta: archivo exportado desde el software corporativo de planta (SAP / sistema interno). Formatos: .xlsx o .csv.
Fuente B — Inventario Proveedor: manifiesto de contenedor provisto por el proveedor externo. Formatos: .xlsx o .csv.
Puede procesar cada fuente por separado o usar Sincronizar ambas para importar las dos en simultáneo. Al finalizar, el semáforo se actualiza automáticamente.

Alertas de Stock
Las alertas se generan automáticamente cuando el stock de una pieza cruza un umbral definido:
Crítico — stock por debajo del mínimo operativo. Requiere acción inmediata.
Advertencia — stock en zona de riesgo. Planificar reposición.
Use Reconocer para confirmar que la alerta fue atendida. Las alertas reconocidas pasan al historial.
